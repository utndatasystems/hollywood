# Data

This directory contains the public Hollywood 200K dataset in two forms.

`Hollywood 200K` refers to `200,000` primary movies (`title.kind_id = 1`).
The full `title` table is larger because the strict IMDb schema also stores
other title kinds in the same table. The released row counts are recorded in
`export_manifest.json`.

- `imdb_csv/`: strict 21-table IMDb/JOB CSV export with headers.
- `hollywood_200k.duckdb`: DuckDB database built from the strict CSVs.
- `export_manifest.json`: source/export metadata from the final exporter run.
- `genre_derivation_summary.json`: genre export summary.
- `source_export_coverage.csv`: field-level source-to-export coverage report.
- `checksums.sha256`: generated by `python scripts/package_dataset.py`.

The public load path uses only the strict 21 tables. Optional generated-world
extras are intentionally not part of this dataset directory.
