# Queries

Final structure-preserving Hollywood SQL workloads:

- `job_light/`: 70 JOB-Light count queries.
- `job/`: 113 JOB queries.
- `job_complex/`: 30 JOB-Complex queries.

The query shapes follow the original workloads. Adaptation changes literals
such as years, string constants, genres, companies, and keywords so the queries
are valid and nonzero on the Hollywood 200K dataset.

Original query provenance and license notes are documented in
`../docs/licensing.md`.
